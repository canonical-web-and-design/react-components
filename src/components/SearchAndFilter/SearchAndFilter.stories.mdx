### External state

If you wish to control the value of the data set via external state you can set
the `externallyControlled` prop and provide an `onChange` method to update the
state and the `value` from state.

<Preview>
  <Story name="External state">
    <SearchAndFilter
      externallyControlled
      onChange={(searchTerm) => {
        console.log("onChange: ", searchTerm);
      }}
    />
  </Story>
</Preview>
